# PRD — AMPH Academy

**Version:** 1.0 | **Status:** Approved | **Last Updated:** 2026-07-02

---

## Product Overview

AMPH Academy is an **Amazon advertising skills training platform** that equips Filipino virtual assistants with specialized Amazon advertising expertise. The platform provides structured courses, interactive tools (Campaign Builder, Bid Elevator, Search Term Triage), gamified learning with badges, and downloadable resources.

**Core Value Proposition:** Master Amazon advertising through hands-on practice with real campaign tools — not just theory.

---

## Target Audience

- Filipino VAs wanting Amazon advertising specialization
- Current PPC specialists wanting advanced campaign optimization skills
- Agency staff needing structured Amazon Ads training
- Self-paced learners preferring interactive tools over video-only courses

---

## User Stories

- US-01: I can register and access structured course content
- US-02: I can use the Campaign Builder to practice campaign structures
- US-03: I can use the Bid Elevator tool for bid optimization practice
- US-04: I can use Search Term Triage for keyword analysis practice
- US-05: I can earn badges as I complete modules and challenges
- US-06: I can track my progress through the curriculum
- US-07: I can test my knowledge with quizzes
- US-08: I can download course resources and templates
